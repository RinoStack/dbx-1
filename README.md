# DBX

Open-source, lightweight, cross-platform database client.

## Supported Databases

- MySQL
- PostgreSQL
- SQLite
- Redis
- MongoDB
- DuckDB
- ClickHouse
- SQL Server

## Features

- Multi-database support
- Query editor
- Visual query builder
- Data export
- Connection management

## Getting Started

```bash
npm install
npm run dev
```
